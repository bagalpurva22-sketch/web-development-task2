#  Laundry Wallah / Laundry Mart Website

This project involves creating a structured web page for a laundry service named **Laundry Wallah** or **Laundry Mart** using HTML. Below are the structural layout requirements for the project.


## Website Structure & Sections

### 1. Header Section
* **Heading:** Display a welcoming heading: `"Welcome to Laundry Wallah / Laundry Mart"`.

### 2. About Section
**Content:** Provide a brief paragraph introducing **Laundry Wallah / Laundry Mart**. 

### 3. Image Section
* **Content:** Insert an image representing a laundry environment.
* **Source:** You can use any appropriate laundry image sourced from the Internet.

### 4. Services Section
* **Heading:** Include a section heading: `"Our Services"`.
* **List:** Use an unordered (`<ul>`) or ordered (`<ol>`) list to enlist the laundry services offered.
* **Examples to include:**
  * Washing and Drying
  * Ironing and Folding
  * Dry Cleaning
  * Stain Removal
  * Express Services
  * Special Garment Care

### 5. Price List Section
* **Heading:** Include a section heading: `"Price List"`.
* **Table:** Create a structured HTML table with a border.

### 6. Book Now Section
* **Heading:** Display a section heading: `"Book Now"`.
* **Form:** Include an interactive form with the following required input fields:
  * **Name** (marked as required)
  * **Email** (marked as required)
  * **Phone Number** (marked as required)
* **Action:**  `Submit` button at the end of the form.

### 7. Footer Section
 **Text to include:** `"Contact us at: info@laundrymart.com"`
